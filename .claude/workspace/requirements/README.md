# Workspace — Requirements

Đặt file BRIEF-{ID}.md tại thư mục này để Agent BA phân tích.

## Quy trình
1. Copy template từ `.claude/templates/BRIEF-template.md`
2. Đặt tại `.claude/workspace/requirements/input/BRIEF-{ID}.md`
3. Điền đầy đủ thông tin
4. Chạy `/analyze-requirements BRIEF-{ID}`


## ID Convention
- BRIEF-00001, BRIEF-00002, ... (5 chữ số, tăng dần)
